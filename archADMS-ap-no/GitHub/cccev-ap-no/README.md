# cccev-ap-no

_For maintanence of CCCEV-AP-NO which is the Norwegian Application Profile of EU's [CCCEV](https://github.com/SEMICeu/CCCEV/)_

For forvaltning av CCCEV-AP-NO som er en norsk applikasjonsprofil av EUs [CCCEV](https://github.com/SEMICeu/CCCEV/). 

Gjeldende versjon blir publisert til: https://data.norge.no/specification/cccev-ap-no

Redaktørens utkast blir publisert til: https://informasjonsforvaltning.github.io/cccev-ap-no/

Ontologien publiseres til: https://data.norge.no/vocabulary/cccevno
